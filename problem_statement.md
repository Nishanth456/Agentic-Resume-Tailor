# Problem Statement: Resume Tailor Agent

## Summary
Job seekers often spend a lot of time rewriting the same resume for each application. This process is repetitive, time-consuming, and easy to get wrong, especially when each job description requires a slightly different emphasis.

## Problem
Most people do not have the time or confidence to manually tailor their resume for every role. As a result, they either:
- submit the same resume for every job,
- miss important keywords,
- or struggle to explain why they are a good fit for a specific role.

This leads to weaker job applications and a lower chance of getting interviews.

## Proposed Solution
Resume Tailor Agent is a simple AI assistant that helps users improve their resume for a target job.

The agent can:
- read a resume and a job description
- identify the most relevant skills and experience
- suggest better bullet points and wording
- highlight missing keywords
- generate a tailored summary or cover letter draft
- explain why the resume is a strong match

## Why This Matters
This is a practical and relatable problem for many people. The agent saves time, improves application quality, and makes career support more accessible.

## Target Users
- students applying for internships
- early-career professionals
- career switchers
- anyone applying to multiple jobs

## Value Proposition
The agent makes job applications easier by turning a manual editing task into a guided, AI-assisted workflow.

## Project Direction
This project will use Google Agent Development Kit (ADK) for multi-agent orchestration, MCP for tool access such as file reading and document processing, and a simple web interface for interaction. The goal is to create a polished prototype that demonstrates how agents can help with real-world productivity tasks.

## Success Criteria
A successful version of this project should:
- understand the role and the resume content
- generate relevant, high-quality tailoring suggestions
- present the output clearly and actionably
- feel useful enough for a real job application workflow
